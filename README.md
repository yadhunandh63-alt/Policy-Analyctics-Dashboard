# Policy-Analyctics-Dashboard|Power BI
Developed an interactive Policy Analytics Dashboard in Power BI to analyze insurance policy performance, customer insights, claims, and premium trends using dynamic visualizations and KPIs.
## Key Features
* Designed an interactive Power BI dashboard with slicers for Status, Payment Status, and Gender to enable dynamic data filtering.
* Created KPI cards to display Total Customers, Total Policies, Total Claim Amount, Average Coverage Amount, and Average Premium Amount.
* Built visualizations to analyze policy distribution by gender, policy type, age group, and policy end year.
* Developed trend analysis for premium amount growth to identify yearly performance patterns.
* Implemented DAX measures and calculated metrics for business insights and reporting.
* Used a clean, modern dashboard layout with charts, cards, and gauges for better decision-making.
